Save

Characters can be saved to your computer using your browser's cookies. These saved characters will only be available if you use the same computer and the same browser program.


Load

Saved character slots are shown below. Click on the appropriate button to load or delete a character.

  • None

Export

You may convert your character into a special string of data which can later be loaded into a character sheet. You can copy this string from below and send it in an email or save it in a text file on your computer.



Characters can also be saved and re-loaded by using a unique link. This can make it easy for you to send your character to a friend. However due to size restrictions, this link will not include custom gear.


Import

Characters can be imported by using a special string of data that is generated when you export a character. Paste the string in the field below and click "Import".
